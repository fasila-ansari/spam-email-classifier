# spam-email-classifier
A simple machine learning project that classifies messages as spam or ham using Python and a Naive Bayes model.
## Overview
This project implements a simple machine learning model that classifies messages as spam or ham (legitimate messages). It demonstrates how natural language processing techniques can be used to detect unwanted messages.
## Technologies Used
- Python
- Pandas
- Scikit-learn
## Method
The model uses a Naive Bayes classifier and a CountVectorizer to convert text messages into numerical features. The trained model then predicts whether a new message is spam or not.
## Example
Input:
Win a free reward now

Output:
Spam
## Purpose
This project was created to practice basic machine learning and text classification techniques using Python.
